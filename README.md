# 🧪 Research Experience

## Research Intern | ICAR – Indian Agricultural Statistics Research Institute (ICAR-IASRI), New Delhi

### 🌱 Project
**Phosphorus Use Efficiency in Wheat using Transcriptomic Analysis**

### Research Summary

Worked on large-scale **Next Generation Sequencing (NGS)** datasets to investigate molecular mechanisms associated with phosphorus use efficiency in wheat through transcriptomic analysis and computational biology approaches.

### Responsibilities

- Performed RNA-Seq based transcriptomic analysis.
- Conducted Differential Gene Expression (DEG) analysis using **limma** and **edgeR**.
- Analysed high-throughput NGS datasets.
- Generated publication-quality volcano plots, PCA plots and heatmaps.
- Conducted Gene Ontology (GO) and KEGG pathway enrichment analysis.
- Performed statistical analysis using R.
- Interpreted biological significance of differentially expressed genes.
- Documented reproducible bioinformatics workflows.
